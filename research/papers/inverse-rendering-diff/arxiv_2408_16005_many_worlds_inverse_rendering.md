# Many-Worlds Inverse Rendering

authors: Ziyi Zhang, Nicolas Roussel, Wenzel Jakob
arxiv: 2408.16005
relevance: Mitsuba-3-era SOTA for differentiable rendering of DISCONTINUOUS configurations - exactly the problem chuck-mcp has when picking plate count, pigment slot, mask topology. Many-worlds treats discrete choices as a superposition and lets gradients flow through all branches simultaneously, then collapses post hoc. Direct alternative to Gumbel-softmax for chuck-mcp's discrete plate-assignment problem.

Algorithmic moves to import:
- For chuck-mcp's discrete pigment assignment per plate: instead of relaxing each choice via Gumbel-softmax, render ALL pigment candidates in parallel as a "superposition" and let the joint loss decide weights.
- Particularly useful when the discrete choice set is small (~36 pigments in chuck-mcp): the cost of evaluating all worlds is bounded.
- Trust-region adjustment: collapse occasionally, then resume superposition.
