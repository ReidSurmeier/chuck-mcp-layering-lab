# Color Me Correctly — Bridging Perceptual Color Spaces and Text Embeddings for Improved Diffusion Generation

arXiv 2509.10058 (Sept 2025, NYCU) — Tsai et al.

## Why this matters for chuck-mcp v2 prompt translation

- Direct precedent: LLM (GPT-4o) parses ambiguous color terms in a natural-language prompt, returns **structured color analysis list** with reference CIELab/RGB codes.
- The LLM's job is *disambiguation*, not optimization. Output is a JSON-like structure: `{token, ambiguous?, basic_color, reference_code}`.
- They benchmark against TintBench (1000 prompts) covering 5 compound-color taxonomies: Blended, Modified, Object, Signature, Abstract — almost exactly the categories Reid uses ("warm-tonal", "vermilion", "deep umber under indigo").
- Conclusion: GPT-4o-class model is sufficient for compound color name → basic color + CIELab anchor. No fine-tuning, no reference images.
